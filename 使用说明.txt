# 预处理工具使用指南

## 项目文件说明

- `yuchuli.py` - 主程序文件
- `getch.py` - 键盘输入处理模块
- `install.sh` - 一键安装脚本
- `requirements.txt` - Python依赖包列表
- `README.md` - 详细说明文档
- `示例数据.xlsx` - 示例Excel文件

## 安装步骤

1. 打开终端并导航到项目目录：
   ```bash
   cd /path/to/待上架预处理
   ```

2. 运行安装脚本：
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

3. 安装完成后，可以在任何位置使用 `yuchuli` 命令启动程序

## 使用方法

1. 在终端中输入 `yuchuli` 启动程序
2. 使用上下箭头键导航菜单选项
3. 按回车键确认选择
4. 按 0 键退出程序（也可在菜单中选择"退出程序(按0键)"选项）

### 处理Excel文件

1. 选择"处理Excel文件"选项
2. 将Excel文件拖拽到终端窗口，或手动输入文件路径
3. 程序将自动处理文件并生成结果

### 查看帮助

选择"查看帮助"选项可查看详细的使用说明和计算规则

## 功能特性

- 🎯 自动识别Excel中的"商品编码"(M列)和"采购价"(S列)
- 🧮 自动计算成本价和基本售价
- 📊 生成标准格式的Excel文件
- 🌈 彩色终端界面，操作直观
- ⌨️ 支持键盘导航操作
- 🚀 一键安装，便捷使用

## 计算规则

- 成本价 = 采购价 + 2
- 基本售价 = ((采购价 + 5) × 2)的整数部分 + 0.99
- 虚拟分类 = "可预售"(固定值)

## 输出格式

生成的Excel文件包含以下列：
1. 商品编码
2. 采购价
3. 成本价
4. 基本售价
5. 虚拟分类

- macOS系统
- Python 3.6或更高版本
- pip包管理器
